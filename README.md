# 🧮 Scientific Calculator (Multi-Theme)

A modern, responsive **Scientific Calculator** with multiple themes, built for fast and accurate mathematical computations.  
Designed with clean UI, smooth interactions, and a customizable user experience.

---

## ✨ Features

- 🧮 Basic arithmetic operations ( + − × ÷ )
- 📐 Advanced scientific functions (sin, cos, tan, log, ln)
- 🔢 Powers, roots, and percentages
- 🎨 Multiple UI themes (Dark, Light, Neon, Gradient, etc.)
- 🌙 Theme switcher with instant UI updates
- ⌨️ Keyboard support for faster input
- 📱 Fully responsive (mobile + desktop)
- ⚡ Fast and lightweight performance

---

## 🎨 Themes

This calculator includes multiple beautiful themes:

- ⚡ Neon Mode — Glowing futuristic style  
- 🌈 Gradient Mode — Color-rich modern design
